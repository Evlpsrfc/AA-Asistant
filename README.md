# AA-Asistant

基于用户需求的APP开发作业——AA助手

## 项目用户

涉及数次不同人员付款、需要AA结账的人群

## 项目需求

- 输入每个人花了多少钱，计算出谁给谁多少
- 中途可能会有谁借谁多少钱
- ~~账簿管理功能~~
- ~~加入汇率~~
- ~~拍照记录~~

## 项目进度

已完成部分：

未完成部分：

历史账单（或称“账簿”）的管理功能、账单单个条目的更多信息查看、账单和人员的单个条目修改、更优化的UI、其他应该完善的部分

## 使用方法

可以事先在“用户”中添加用户，也可以通过“账单”的“记一笔”按钮自动添加。点击“账单”的“结账”按钮即可显示谁应该给谁多少钱。点“确认结账”会删除当前账单（以后会将删除操作变为存储到“历史”中）。
